# FPV Drone Strikes Lebanon Dataset

This repository indexes a curated and constantly updated set of FPV drone strike videos related to Hezbollah activity on the Lebanon-Israel front. The video files and preview thumbnails are stored in S3, while this repo provides a lightweight catalog with stable browser links.

## Storage

- Bucket: `s3://fpv-drone-strikes-lebanon-dataset`
- Video prefix: `videos/`
- Thumbnail prefix: `thumbnails/`
- Video count: 148 MP4 files
